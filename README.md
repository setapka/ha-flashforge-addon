# Flashforge Adventurer 5M Control Panel - Home Assistant Addon

Аддон для управления 3D-принтером Flashforge Adventurer 5M с прошивкой Klipper/Moonraker.

## Установка

### Шаг 1: Удалите старый репозиторий (ВАЖНО!)

Если вы ранее добавляли этот репозиторий, его нужно удалить:

1. Откройте Home Assistant
2. Перейдите в **Настройки** → **Дополнения** → **Магазин дополнений**
3. Нажмите на **три точки** (справа сверху)
4. Выберите **Репозитории**
5. Найдите запись с URL `https://raw.githubusercontent.com/setapka/ha-flashforge-addon/main/` или похожую
6. Нажмите на неё → **Удалить**
7. **Перезагрузите страницу браузера** (F5) для очистки кэша

### Шаг 2: Добавьте новый репозиторий

1. Снова нажмите на **три точки** → **Репозитории**
2. Введите URL: `https://github.com/setapka/ha-flashforge-addon`
3. Нажмите **Добавить**

### Шаг 3: Установите аддон

1. В магазине дополнений найдите **Flashforge Adventurer 5M Control Panel**
2. Нажмите на него
3. Нажмите **Установить**
4. После установки перейдите на вкладку **Конфигурация**
5. Введите IP адрес вашего принтера
6. Нажмите **Сохранить** и **Запустить**

## Файлы репозитория

- `config.yaml` - конфигурация аддона
- `Dockerfile` - образ Docker
- `repository.json` - описание репозитория
- `rootfs/` - файловая система аддона
